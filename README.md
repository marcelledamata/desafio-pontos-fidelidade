📌 Desafio: Sistema de Pontos de Fidelidade

📖 Sobre o projeto

Este projeto foi desenvolvido como parte de um desafio do programa Estartando Devs.
O objetivo é calcular os pontos de fidelidade de um cliente com base nas suas compras em uma loja online.

🧠 Regras do sistema

Cada compra possui um valor e uma categoria, e os pontos são calculados da seguinte forma:

A cada R$ 10, o cliente ganha 1 ponto (valores decimais são ignorados)
Compras com valor menor que R$ 10 não geram pontos
Multiplicadores por categoria:
📱 eletronico → multiplica os pontos por 3
📚 livro → multiplica os pontos por 2
🍎 alimento → não altera os pontos (multiplicador 1)
🎁 Bônus de fidelidade

Se o total de pontos ultrapassar 100 pontos, o cliente recebe um bônus de 20% sobre o total final, arredondado para baixo.

💻 Tecnologias utilizadas
JavaScript (ES6)
Git e GitHub

Praticar lógica de programação, manipulação de arrays e objetos, além de versionamento de código com Git.
